STL常用容器和函数

vector

元素访问

1、下标

2、迭代器

vector v; 
for (vector::iterator it = v.begin(); it != v.end(); it++)
    printf("% d", *it);

常用函数

push_back(x):尾部添加元素

pop_back():删除vector的尾元素

size():元素个数

clear():清空所有元素

insert(it, x):向任意迭代器it处插入一个元素x
v.insert(v.begin() + 2, -1); //将-1插入v[2]的位置

erase():删除单个元素、删除一个区间内的所有元素

//erase(it): 删除迭代器为it处的元素
//v = [5 6 7 8 9],删除8
v.erase(v.begin() + 3); //v.begin()对应v[0]，所以8不是对应v.begin() + 4

//erase(first, end):删除[first, last)内的所有元素
erase(v.begin() + 1, v.begin() + 4); //删除v[1],v[2],v[3]

常见用途

(1) 存储数据
(2) 用邻接表存储图

set

内部自动有序且不含重复元素的容器
set内的元素自动递增排序且自动去除了重复元素

元素访问

只能通过迭代器
除开vector和string之外的容器都不支持*(it + i)的访问方式，只能枚举

set st;
for(set::iterator it = st.begin(); it != st.end(); it++)
    printf("%d ", *it)

常用函数

insert(x):插入容器

find(value):返回set中对应值为value的迭代器
‘printf(“%d”, *(st.find(2)); //输出2’

erase():删除单个元素、删除一个区间内的所有元素

//erase(it/value)it为所需要删除元素的迭代器，结合find()来使用; value为元素值
//st = [100 200 300],删除100
st.erase(st.find(100)); 
st.erase(100);

//erase(first, end):删除[first, last)内的所有元素
set::iterator it = st.find(30);
st.erase(it, st.end()); //删除元素30至set末尾之间的元素

size():获得元素个数

clear():清空set

常见用途

最主要作用就是自动去重并按元素升序排序

注：set中元素唯一，若需要处理不唯一的情况，则需要使用multiset。另外，c++ 11标准中还增加了unordered_set以散列代替set内部的红黑树，使其可以用来处理只去重不排序的需求，速度比set要快得多

string

内容访问

(1)下标访问
一般来说，可以直接像字符数组那么去访问string

若要读入输出整个字符串，则只能用cin、cout

也可用c_str()将string转换为字符数组输出
printf("%s", str.c_str())

(2)迭代器访问

string str = "abcd";
for (string::iterator it = str.begin(); it != str.end(); it++)
    printf("%c", *it);
//输出结果a b c d

string和vector一样，支持直接对迭代器加减数字，如str.begin() + 3

常用函数

length()/size():返回长度

insert()

//insert(pos, string), 在pos号位置插入字符串string
string str = "abcxyz". str2 = "opq";
str.insert(3, str2); 
//输出abcopqxyz


//insert(it, it2, it3), it为原字符串的欲插入位置
//it2和it3为待插字符串的首尾迭代器，用来表示串[it2,it3)将被插在it的位置上
string str = "abcxyz". str2 = "opq";
str.insert(str.begin() + 3, str2.begin(), str2.end());
//输出abcopqxyz

erase()

string str = "abcdefg;

//erase(it), 删除单个元素
str.erase(str.begin() + 4); //删除e

//erase(first, last), 删除区间内元素, 即删除[first, last)
str.erase(str.begin() + 2, str.end() - 1);  //删除cdef

//erase(pos, length), pos为开始删除起始位置，length为删除的字符个数
str.erase(3, 2);    //删除de

clear():清空string的内容

substr(pos, len):返回从pos开始，长度为len的字符串

string::npos
是个常数，值为-1，但由于是unsigned_int类型，因此实际上也可认为是unsigned_int类型的最大值(即可认为是-1 或 4294967295)。string::npos用以作为find函数失配时的返回值。

find():
find(str2):当str2是str的字串时，返回其在str中第一次出现的位置；如果不是字串，返回string::npos
find(str2, pos):从str的pos位置开始匹配str2，返回值同上

replace():
replace(pos, len, str2):把str从pos位置开始、长度为len的字串替换为str2
replace(it1, it2, str2):把str的迭代器[it1, it2)范围的字串替换为str2

map

元素访问

map mp;
mp['m'] = 20; 
mp['m'] = 10;   //覆盖
//下标访问
printf("%d ", mp['m']);


//迭代器访问
for (map::iterator it = mp.begin(); it != mp.end(); it++)
    printf("%c %d", it->first, it->second);

map会以键从小到大的顺序自动排序，这是由于map内部使用红黑树实现（set也是）在建立映射的过程中会自动实现排序

常用函数

find(key):返回键为key的迭代器
printf("%d", mp.find('a')->second);

erase()

//erase(it),it为需要删除元素的迭代器
mp.erase(mp.find('a'));

//erase(key),key为欲删除的映射的键
mp.erase('a');

//erase(first, last), 删除区间[first, last)
mp.erase(mp.find('a'), mp.end());

size():映射对数

clear(): 清空map

常见用途

1、建立映射关系
2、判断大整数或其他类型数据是否存在，可把map当bool数组使用

map的键和值是唯一的，如果需要一个键对应多个值，就只能用multimap。c++ 11标准中还增加了unordered_map，以散列代替map内部的红黑树实现，使其可以只映射而不按key值排序，速度要快得多

queue

元素访问

由于队列本身的限制，所以只能用front()来访问队首元素，back()来访问队尾元素

常用函数

front(),back(): 访问队首、队尾元素

pop(): 队首元素出队列

empty(): 队列是否为空，空则返回true

size(): 元素个数

常见用途

广度优先搜索
STL中的队列还有双端队列(deque)和优先队列(priority_queue),前者首尾皆可插入和删除，后者使用堆实现默认将当前队列最大元素置于队首

priority_queue

元素访问

优先队列没有front和back，而只能通过top()来访问队首元素，也就是优先级最高的元素

常用函数

push(x): x入队

top(): 获得队首元素

pop(): 队首元素出队

empty(): 队列空则返回true

size(): 元素个数

优先级设置

1、基本数据结构的优先级设置(int、char、double等)
vector 用来承载底层数据结构堆的容器，如果第一个参数为double或char，则此处只需填写vector或vector
第三个参数less则是对第一个参数的比较类，less表示数字大的优先级越大，greater表示数字小的优先级越大

priority_queue q;
priority_queue, less > q;

2、结构体的优先级设置，重载小于号 <

struct fruit {
    int price;
    friend bool operator < (fruit f1, fruit f2) {
        return f1.price < f2.price; //价格高的优先级高
    //  return f1.price > f2.price; 价格低的优先级高
    }
};
priority_queue q;

可以发现优先级队列的这个函数与sort中的cmp的效果是相反的,在cmp中’return f1.price > f2.price;’那么价格按从高到低排列

优先级设置函数也可写在结构体外

struct cmp {
    bool operator () (fruit f1, fruit f2) {
        return f1.price > f2.price; 
    }
};
priority_queue, cmp> q;

若结构体内的数据较为庞大(例如出现了字符串或数组),建议使用引用来提高效率，此时比较类的参数中需要加上”const”和”&”

friend bool operator < (const fruit &f1, const fruit &f2) {
    return f1.price < f2.price;  
}

bool operator () (const fruit &f1, const fruit &f2) {
    return f1.price > f2.price; 
}

常见用途

可解决一些贪心问题，也可对Dijkstra算法进行优化(因为优先队列的本质是堆)

stack

元素访问

由于栈本身的限制，所以只能通过top()来访问栈顶元素

常见函数

push(x):入栈

top():获得栈顶元素

pop():弹出栈顶元素

empty():空则返回true

size():元素个数

常见用途

模拟实现一些递归，防止程序对栈内存的限制导致程序运行出错

pair

pair的构建

//1
pair p;
p.first = "hh";
p.second = 5;

//2
p = make_pair("hh", 5);

//3
p = pair("hh", 5);

常用函数

比较操作数时，先以first的大小作为标准，只有当first相等时才去判别second

常见用途

1、用来代替二元结构体及其构造函数
2、作为map的键值对来进行插入

map mp;
mp.insert(make_pair("hh", 5));
mp.insert(pair("hh", 5));

algorithm头文件下的常用函数

max,min,abs,swap

max(x, y):返回x，y中的最大值

min(x, y):返回x，y中的最小值

abs(x):返回x的绝对值(x为整型)，浮点型用fabs

swap(x, y):交换x，y值

reverse

reverse(it, it2): [it, it2)之间的元素反转

int a[10] = {0, 1, 2 ,3, 4, 5};
reverse(a, a + 4);  //变成3 2 1 0 4 5


string str = "abcdefghi";
reverse(str.begin() + 2, str.begin() + 6);
//变成ab fedc ghi

next_permutation

给出一个序列在全排列中的下一个序列，到达最后一个会返回false

int a[10] = {1, 2, 3};
do {
    printf("%d %d %d", a[0], a[1], a[2]);
}while(next_permutation(a, a + 3)); 
//输出123 132 213 231 312 321

fill

fill():数组或容器某一段区间内赋为某个相同的值，和memset不同，可为范围内任意值
fill(a, a + 5, 11); //将a[0] ~ a[4]赋值为11

sort

sort()：
sort(首元素地址(必填)，尾元素地址的下一个地址(必填)，比较函数(非必填);，默认递增

//从大到小排序
bool cmp(int a, int b) {
    return a > b;
}
sort(a, a + 4, cmp);

//结构体排序
struct node {
    int x, y;
}ssd[10];

bool cmp(node a, node b) {
    if (a.x != b.x) return a.x > b.x;
    return a.y < b.y;   //x递增，x相等情况下，y递减
}

sort(ssd, ssd + 3, cmp);

//容器排序
//基本类似，只有vector、string、deque可以使用sort
//set、map用红黑树实现本身有序，故不允许使用sort

lower_bound、upper_bound

lower_bound(first, last, val)用来寻找在数组或容器的[first, last)范围内第一个大于等于val的元素的位置，数组则返回该位置的指针，容器则返回该位置的迭代器

upper_bound(first, last, val) 返回第一个大于val的元素的位置，其它同上

若数组或容器中没有需要寻找的元素，则返回可以插入该元素的位置的指针或迭代器

int a[10] = {1, 2, 2, 3, 3, 3, 5, 5, 5, 5};
//寻找-1
int* lowerPos = lower_bound(a, a + 10, -1);
int* upperPos = upper_bound(a, a + 10, -1);
printf("%d %d\n", lowerPos - a, upperPos - a);
//输出0, 0
//要获得待查元素的下标，令返回值减去数组首地址

//寻找1
lowerPos = lower_bound(a, a + 10, 1);
upperPos = upper_bound(a, a + 10, 1);
printf("%d %d\n", lowerPos - a, upperPos - a);
//输出0, 1 

//寻找3
lowerPos = lower_bound(a, a + 10, 3);
upperPos = upper_bound(a, a + 10, 3);
printf("%d %d\n", lowerPos - a, upperPos - a);
//输出3, 6

//寻找6
lowerPos = lower_bound(a, a + 10, 6);
upperPos = upper_bound(a, a + 10, 6);
printf("%d %d\n", lowerPos - a, upperPos - a);
//输出10, 10
